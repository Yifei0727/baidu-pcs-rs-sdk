# 百度网盘远程文件移动/重命名

## 使用场景

- 在网盘内移动文件或目录到新位置
- 对网盘中的文件或目录重命名

## 命令格式

```bash
baidu-pan-cli-rs mv <源路径> <目标路径>
# 别名
baidu-pan-cli-rs rename <源路径> <目标路径>
```

## 参数说明

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `<源路径>` | 必填 | 网盘中要移动/重命名的源文件/目录路径 | `/旧目录/file.txt` |
| `<目标路径>` | 必填 | 网盘中的目标路径（含新文件名） | `/新目录/file.txt` |

## 注意事项

- 源路径和目标路径均为**百度网盘远程路径**
- 重命名时目录不变，仅修改末尾的文件名部分
- 移动时目标目录必须已存在

## 示例

```bash
# 移动文件
baidu-pan-cli-rs mv /临时/report.pdf /归档/2024/report.pdf

# 重命名文件
baidu-pan-cli-rs rename /我的文件/旧名.txt /我的文件/新名.txt

# 移动并重命名目录
baidu-pan-cli-rs mv /项目/draft /项目/release
```
