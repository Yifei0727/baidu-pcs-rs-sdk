# 百度网盘目录列表

## 使用场景

- 查看远程目录中有哪些文件
- 确认文件是否已存在于网盘
- 浏览远程目录树结构

## 命令格式

```bash
baidu-pcs-cli-rs ls <远程路径> [-r]
# 别名
baidu-pcs-cli-rs list <远程路径> [-r]
```

## 参数说明

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `<远程路径>` | 必填 | 百度网盘上的目录路径 | `/我的文件/照片` |
| `-r` / `--recursive` | 可选 | 递归列出所有子目录内容 | `-r` |

## 输出格式

每行输出格式为：

```
<类型>  <大小(字节)>  <文件名>  <完整路径>  <文件ID>
```

- 类型：`d` 表示目录，`-` 表示文件

## 示例

```bash
# 列出根目录
baidu-pcs-cli-rs ls /

# 列出指定目录
baidu-pcs-cli-rs ls /我的文件/项目

# 递归列出目录及所有子目录
baidu-pcs-cli-rs ls /我的文件 -r
```
