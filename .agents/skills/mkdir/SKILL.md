# 百度网盘远程目录创建

## 使用场景

- 在网盘中创建新目录
- 一次性创建多个目录
- 创建嵌套目录（父目录不存在时自动创建）

## 命令格式

```bash
baidu-pcs-cli-rs mkdir <远程路径>... [-p]
# 别名
baidu-pcs-cli-rs md <远程路径>... [-p]
```

## 参数说明

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `<远程路径>...` | 必填（支持多个） | 要创建的网盘目录路径，空格分隔 | `/项目 /归档` |
| `-p` / `--parents` | 可选 | 父目录不存在时自动逐级创建 | `-p` |

## 注意事项

- 不加 `-p` 时，若父目录不存在则会创建失败
- 支持同时传入多个路径批量创建

## 示例

```bash
# 创建单个目录
baidu-pcs-cli-rs mkdir /我的文件/新项目

# 创建嵌套目录（自动创建父目录）
baidu-pcs-cli-rs mkdir /我的文件/项目/子目录 -p

# 批量创建多个目录
baidu-pcs-cli-rs mkdir /项目 /归档 /临时
```
