# 百度网盘增量备份

## 使用场景

- 定期将本地目录备份到网盘（增量，不重复上传）
- 首次全量备份后，后续仅同步新增文件
- 结合 `--remove-source` 将本地文件归档迁移到云端

## 命令格式

```bash
baidu-pan-cli-rs backup <本地路径> <远程路径> [--remove-source]
```

## 参数说明

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `<本地路径>` | 必填 | 本地源文件或目录路径 | `~/documents` |
| `<远程路径>` | 必填 | 网盘目标目录路径 | `/备份/documents` |
| `--remove-source` | 可选 | 备份成功后删除本地源文件 | `--remove-source` |

## 与 `tx` 命令的区别

| | `backup` | `tx` |
|-|---------|------|
| 已存在文件 | **跳过** | 覆盖上传 |
| 适用场景 | 增量备份 | 全量同步/覆盖 |

## 注意事项

- 文件存在性通过**文件名**判断，若内容变更但文件名相同则不会更新（如需覆盖更新请使用 `tx`）
- `--remove-source` 在备份成功后删除本地源文件，请确认后再使用

## 示例

```bash
# 增量备份本地目录到网盘
baidu-pan-cli-rs backup ~/documents /备份/documents

# 备份并删除本地源（归档到云端）
baidu-pan-cli-rs backup ~/旧项目 /归档/旧项目 --remove-source
```
