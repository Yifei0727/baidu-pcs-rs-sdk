# 百度网盘远程文件复制

## 使用场景

- 在网盘内部复制文件到另一目录
- 备份网盘中的某个文件/目录
- 将文件从一个网盘路径复制到另一个路径

## 命令格式

```bash
baidu-pcs-cli-rs cp <源路径> <目标路径>
# 别名
baidu-pcs-cli-rs copy <源路径> <目标路径>
```

## 参数说明

| 参数 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `<源路径>` | 必填 | 网盘中要复制的源文件/目录路径 | `/我的文件/report.pdf` |
| `<目标路径>` | 必填 | 网盘中的目标路径（含文件名） | `/备份/report.pdf` |

## 注意事项

- 源路径和目标路径均为**百度网盘远程路径**，不涉及本地文件
- 目标路径需要包含文件名，而非仅指定目录

## 示例

```bash
# 复制单个文件
baidu-pcs-cli-rs cp /我的文件/report.pdf /备份/report.pdf

# 复制目录
baidu-pcs-cli-rs copy /项目/v1 /项目/v1-backup
```
